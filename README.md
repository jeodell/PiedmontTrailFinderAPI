# PiedmontTrailFinderAPI

> Backend API for Piedmont Trail Finder application.

## Usage

Rename "config/config_copy.env" to "config/config.env" and update the values.

## Install Dependencies

```
npm i
```

## Run App

```
// Run in dev mode
npm run dev

// Run in prod mode
npm run prod
```
